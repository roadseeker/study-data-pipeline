# data

Pipeline Lab의 로컬 데이터 작업 영역이다.

이 폴더는 실습 입력 데이터와 로컬 런타임 산출물을 함께 담지만, Git에는 재현에 필요한 설명 파일과 샘플 데이터만 남긴다. 실제 처리 결과, 체크포인트, 로그성 산출물은 로컬 환경에서 재생성 가능한 대상으로 본다.

## 주요 하위 폴더

| 폴더 | 역할 |
|------|------|
| sample/ | 실습 재현을 위한 소량의 기준 샘플 데이터 |
| settlement/ | Spark 배치 실습에서 읽는 정산 입력 파일 영역 |
| lakehouse/ | Week 5 이후 Spark/Delta Lake 기반 레이크하우스 저장 영역 |
| flink/ | Flink JAR, 체크포인트, 세이브포인트 등 로컬 런타임 산출물 |
| nifi/ | NiFi 정산 CSV 수집 실습에서 생성·처리되는 로컬 파일 |

## Git 관리 기준

- `data/sample/**`는 재현용 샘플 데이터로 Git 추적 대상이다.
- `data/lakehouse/**`의 실제 Delta 파일, 체크포인트, 품질 리포트는 Git 추적 대상이 아니다.
- 각 폴더의 `README.md`는 폴더 역할을 설명하기 위해 Git에 남긴다.
